HTML
langage de description de pages utilisé pour le Web. Ce langage se compose de balises qui seront interprétées par le navigateur du poste client, de texte et d'images qui seront affichées.
Navigateur
Il s'agit d'un programme interprétant le code HTML pour afficher une page Web. Les deux navigateurs les plus communément utilisés sont :
  • Mozilla Firefox
  • Internet Explorers
Web
Ensemble de sites mettant à la disposition des internautes une multitude de ressources (textes, images, sons...). Chaque page du web est munie de liens permettant la navigation.
